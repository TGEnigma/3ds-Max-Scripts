Step 1: Get QuickBMS (http://aluigi.org/papers.htm#quickbms)
Step 2: Use the QuickBMS script on the DAT files to extract them, the folders are created automatically so you can select the same folder as where the DAT files are stored.
Step 3: Place wimgt.exe in your 3ds Max root directory (eg C:\Program Files\Autodesk\3ds Max 2010)
Step 4: Run the MaxScript on the MDB files extracted from the DATs
